## ✨ Key Features

### 🎨 Design

- Pixel-perfect responsive design (mobile-first approach)
- Smooth animations and transitions
- Glassmorphism effect on sticky navbar (dark mode)
- Service cards with gradient backgrounds and hover effects

### ⚡ Performance

- Lazy-loaded images (`loading="lazy"`)
- Async Font Awesome loading
- Optimized CSS (no redundancy, efficient selectors)
- Minified and production-ready code

### ♿ Accessibility

- WCAG 2.1 compliant
- Semantic HTML5 elements
- ARIA labels and roles
- Keyboard navigation support
- Focus visible states
- Prefers-reduced-motion support

### 🌓 User Experience

- Dark/Light mode with system preference detection
- Persistent theme selection (localStorage)
- Smooth scroll to sections
- Mobile menu with body scroll lock
- Form validation
